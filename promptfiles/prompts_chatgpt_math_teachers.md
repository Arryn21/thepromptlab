## THE RTFC FORMULA (use this for ANY prompt)
**R**ole + **T**ask + **F**ormat + **C**riteria

> "Act as an experienced [GRADE] [SUBJECT] teacher (Role). Create a [TASK] (Task). Format it as [FORMAT] (Format). Make sure it is [CRITERIA] (Criteria)."

## 1. WORKSHEET GENERATOR

### Basic Worksheet
> "Act as an experienced high school [SUBJECT] teacher. Create a [NUMBER]-question practice worksheet on [TOPIC] for [GRADE] students. Format it as a clean printable with space between each problem for student work, and include a full answer key with step-by-step worked solutions on a separate page. Make all answers integers only and align to Common Core standards."

**Fill in:** SUBJECT (Algebra 1, Geometry, Pre-Calculus, Statistics...), NUMBER (10, 15, 20), TOPIC, GRADE

### Example:
> "Act as an experienced high school Algebra 1 teacher. Create a 12-question practice worksheet on solving two-step equations for 9th graders. Format it as a clean printable with space between each problem for student work, and include a full answer key with step-by-step worked solutions on a separate page. Make all answers integers only and align to Common Core 8.EE.C.7."

## 2. DIFFERENTIATED WORKSHEET (3 LEVELS)

> "I teach [SUBJECT] to a mixed-ability [GRADE] class. Create THREE versions of a [NUMBER]-problem practice set on [TOPIC]:
> - **Version A (Below Grade Level):** [EASIER DESCRIPTION — e.g., 'basic substitution, no negatives']
> - **Version B (On Grade Level):** [STANDARD DESCRIPTION — e.g., 'substitution and elimination, some negative numbers']
> - **Version C (Above Grade Level):** [HARDER DESCRIPTION — e.g., 'all methods, word problems, fractional coefficients']
> Include a complete answer key for each version. Format each as a separate printable section."

### Example:
> "I teach Algebra 1 to a mixed-ability 9th grade class. Create THREE versions of a 6-problem practice set on solving systems of linear equations:
> - Version A: substitution only, positive integer solutions, one-step setup
> - Version B: substitution and elimination, some negative solutions
> - Version C: all three methods, word problems requiring students to set up the system, some fractional coefficients
> Include a complete answer key for each version. Format each as a separate printable section."

## 4. WORD PROBLEM CREATOR

> "Write [NUMBER] engaging word problems about [MATH TOPIC] for [GRADE] students. Use real-world contexts that teenagers find relevant: [CHOOSE: social media, sports, streaming services, part-time jobs, gaming, food, fashion, music, travel]. Each problem must require students to [MATH SKILL — e.g., 'set up and solve a linear equation']. Include complete solutions."

### Example:
> "Write 6 engaging word problems about linear equations for 8th graders. Use real-world contexts involving social media, sports, and part-time jobs. Each problem must require students to set up and solve a linear equation. Include complete solutions."

## 5. LESSON PLAN GENERATOR

> "Design a [LENGTH]-minute [SUBJECT] lesson plan for [GRADE] students on [TOPIC]. Include:
> - Learning objectives aligned to Common Core [STANDARD if known]
> - Warm-up activity ([X] min)
> - Direct instruction ([X] min)
> - Guided practice ([X] min)
> - Independent practice ([X] min)
> - Exit ticket ([X] min)
> List all materials needed. Write at a level where a substitute teacher could follow it."

### Example (60-minute):
> "Design a 60-minute Geometry lesson plan for 10th graders on the Pythagorean theorem and its converse. Include: objectives aligned to Common Core 8.G.B.7, warm-up (5 min), direct instruction (15 min), guided practice (20 min), independent practice (15 min), exit ticket (5 min). List all materials needed."

## 6. EXIT TICKET BANK

> "Create a bank of [NUMBER] exit ticket questions for a [GRADE] [SUBJECT] unit on [TOPIC]. Include:
> - [X] conceptual questions (explain, compare, identify, describe)
> - [X] procedural questions (calculate, graph, solve, simplify)
> - [X] application questions (real-world scenarios, word problems)
> Each question should take no more than 3 minutes to answer. Provide answers for all questions."

### Example:
> "Create a bank of 15 exit ticket questions for a 10th grade Algebra 2 unit on exponential functions. Include 5 conceptual, 5 procedural, and 5 application questions. Each should take ≤3 minutes. Provide answers for all 15."

## 9. REAL-WORLD CONNECTIONS

> "I'm teaching [TOPIC] to [GRADE] students. Give me 5 compelling, real-world examples of where [TOPIC] is used in everyday life or careers that would be relevant and interesting to a [GRADE] student. For each example, suggest one follow-up activity or discussion question I can use in class."

### Example:

## POWER TIPS

**Tip 1 — Layer your prompts:**
Don't try to get everything in one prompt. Start with the worksheet, then:
- "Now make problem 5 harder"
- "Add a word problem about basketball"
- "Now give me the answer key with full working"

**Tip 2 — Specify your standards:**
Add "aligned to California Common Core [STANDARD]" to any prompt for standards-aligned output.

**Tip 3 — Use Code Interpreter for math accuracy:**
In ChatGPT, when Code Interpreter is active, it runs actual calculations — much more accurate than relying on its memory. Enable it for answer key generation.

**Tip 4 — Save your best prompts:**
Create a Google Doc called "My AI Prompts" and paste the ones that work. You'll reuse them all year.

**Tip 5 — Always review before printing:**
AI can make math errors, especially on multi-step problems. Always check your answer key before distributing to students.
